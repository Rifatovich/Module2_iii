# Веб-краулер для сбора статистики сайта 

Собирает:
- общее количество обработанных страниц
- общее количество всех ссылок
- количество уникальных внутренних страниц
- количество неработающих ссылок
- количество внутренних поддоменов
- общее количество ссылок на внешние ресурсы
- количество уникальных внешних ресурсов
- количество уникальных ссылок на файлы .doc/.docx/.pdf

## Все параметры запуска

```bash
python main.py --url <URL> --depth <число> --max-pages <число> --delay <секунды> --output <путь/название> --browser-path <путь к браузеру>
